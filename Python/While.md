## While

#### while문 쓰기

#### while문

- 조건이 참인 경우 계속 실행하는 반복문

  ```python
  while selected not in ['가위', '바위', '보']:
      selected = input('가위, 바위, 보 중에 선택하세요>')
  ```

- for 반복문으로 작성한 코드는 while 반복문으로 작성 할 수 있다.



### break, continue

#### break

- 반복문을 종료시키는 기능

#### continue

- 반복문의 나머지 부분을 보지 않고, 반복문의 처음으로 돌아가는 기능
